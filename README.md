
# 🦐 小聋虾链游 (Shrimp RPG)

一个基于币安链的完整 RPG 链游！

## 🎮 游戏特色

- **角色养成**：小聋虾角色升级
- **战斗系统**：PVE 和 PVP 战斗
- **装备系统**：武器、盔甲、饰品
- **公会系统**：创建公会、公会战
- **NFT 资产**：角色、装备都是 NFT
- **Play-to-Earn**：玩游戏赚取代币
- **DAO 治理**：玩家参与游戏治理
