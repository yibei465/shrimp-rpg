
# 🎮 小聋虾链游 - 游戏玩法

## 🦐 游戏世界观

在加密海洋深处，有一个神秘的虾王国。你是一只勇敢的小聋虾，为了保卫虾王国，你需要不断成长、战斗、收集装备！

## ⚔️ 核心玩法

### 1. 角色创建
- 连接钱包，创建你的小聋虾 NFT 角色
- 选择职业：战士、法师、射手
- 每个职业有不同的属性和技能

### 2. 角色养成
- 通过战斗获得经验值升级
- 升级提升属性：力量、智力、敏捷
- 学习新技能

### 3. 战斗系统
- **PVE 战斗**：和怪物战斗，获得奖励
- **PVP 战斗**：和其他玩家对战
- **公会战**：公会之间的大规模战斗

### 4. 装备系统
- 装备分为：武器、盔甲、饰品
- 装备可以升级、强化、合成
- 稀有装备有特殊属性

### 5. 公会系统
- 创建或加入公会
- 公会任务和奖励
- 公会商店和技能

### 6. 代币经济
- **SHRIMP 代币**：游戏主代币
- 战斗奖励、任务奖励
- 质押、治理、投票

## 🎯 游戏目标

1. 升级你的小聋虾角色
2. 收集强力装备
3. 加入或创建强大的公会
4. 成为虾王国的最强者！

## 💰 Play-to-Earn

- 战斗胜利获得 SHRIMP 代币
- 稀有装备可以交易
- 参与治理获得奖励
- 质押代币获得收益
